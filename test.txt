# Тестовые наборы для GraphVisualizer

## 1. Неориентированный граф (симметричная матрица)

### Матрица:
```
0 1 0
1 0 1
0 1 0
```

### Список смежности:
```
0: →1
1: →0 →2
2: →1
```

**Ожидаемый результат:**
- Нет стрелок
- Один вес на каждом ребре (даже если вес = 1.0)
- Нет дублирования весов

---

## 2. Ориентированный граф (несимметричная матрица)

### Матрица:
```
0 2 6 0 0
2 0 3 5 0
6 3 0 1 0
0 5 1 0 4
0 0 0 4 0
```

### Список смежности:
```
0: →1(2) →2(6)
1: →0(2) →2(3) →3(5)
2: →0(6) →1(3) →3(1)
3: →1(5) →2(1) →4(4)
4: →3(4)
```

**Ожидаемый результат:**
- Стрелки на всех ориентированных рёбрах
- Все веса отображаются
- Нет дублирования весов

---

## 3. Ориентированный граф с двусторонними рёбрами (одинаковый вес)

### Матрица:
```
0 2 0
2 0 3
0 3 0
```

### Список смежности:
```
0: →1(2)
1: →0(2) →2(3)
2: →1(3)
```

**Ожидаемый результат:**
- Рёбра 0↔1 и 1↔2 рисуются как неориентированные (без стрелок)
- Один вес на каждом ребре (не дублируется)
- Остальные рёбра с стрелками, если они односторонние

---

## 4. Ориентированный граф с двусторонними рёбрами (разные веса)

### Матрица:
```
0 2 0
5 0 3
0 4 0
```

### Список смежности:
```
0: →1(2)
1: →0(5) →2(3)
2: →1(4)
```

**Ожидаемый результат:**
- Все рёбра с стрелками (разные веса = разные рёбра)
- Веса: 0→1(2), 1→0(5), 1→2(3), 2→1(4)
- Нет дублирования весов

---

## 5. Смешанный граф (часть неориентированная, часть ориентированная)

### Матрица:
```
0 2 0 0
2 0 0 5
0 0 0 0
0 0 0 0
```

### Список смежности:
```
0: →1(2)
1: →0(2) →3(5)
2: 
3: 
```

**Ожидаемый результат:**
- Ребро 0↔1 рисуется как неориентированное (без стрелок, один вес)
- Ребро 1→3 рисуется с стрелкой
- Нет дублирования весов

---

## 6. Граф с весами = 1.0

### Матрица:
```
0 1 1

1 1 0
```

### Список смежности:
```

1: →0 →2

**Ожидаемый результат:**
- Нет стрелок (неориентированный граф)
- Все веса отображаются (1.0)
- Нет дублирования весов

## 8. Граф с одним ребром в обе стороны (разные веса)

## 9. Граф с одним ребром в обе стороны (одинаковый вес)

### Матрица:
```

```

```
0: →1(3)

**Ожидаемый результат:**

## 10. Ручное построение графа

**Шаги:**
1. Добавить вершины: V0, V1, V2
2. Добавить ребро: 0 → 1, вес 2.5, ориентированное
3. Добавить ребро: 1 → 0, вес 2.5, ориентированное
4. Добавить ребро: 1 → 2, вес 3.0, ориентированное

**Ожидаемый результат:**
- Ребро 0↔1 рисуется как неориентированное (без стрелок, один вес 2.5)
- Ребро 1→2 рисуется с стрелкой, вес 3.0
- Нет дублирования весов

---

## 11. Проверка алгоритма Дейкстры

### Матрица:
```
0 2 6 0 0
2 0 3 5 0
6 3 0 1 0
0 5 1 0 4
0 0 0 4 0
```

**Тесты:**
- Путь от 0 до 4: должен быть 0 → 1 → 3 → 4 (длина 11)
- Путь от 0 до 2: должен быть 0 → 1 → 2 (длина 5)
- Путь от 2 до 4: должен быть 2 → 3 → 4 (длина 5)

**Ожидаемый результат:**
- Путь подсвечивается зелёным цветом
- Вершины пути подсвечиваются золотым цветом
- Программа не падает

---

## 12. Проверка перемещения вершин

**Шаги:**
1. Построить любой граф
2. Переместить вершину

**Ожидаемый результат:**
- Рёбра перерисовываются корректно
- Веса не дублируются
- Стрелки отображаются правильно
- Программа не падает

---

## Примечания

- Все тесты должны проходить без дублирования весов
- Для неориентированных графов стрелки не должны отображаться
- Для ориентированных графов стрелки должны отображаться на всех односторонних рёбрах
- Для двусторонних рёбер с одинаковым весом должна быть одна линия без стрелок
- Для двусторонних рёбер с разными весами должны быть две линии со стрелками
